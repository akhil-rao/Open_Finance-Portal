An intelligent, API-first Open Finance portal designed for banks, fintechs, and developers to explore mock KYC, KYB, Risk, and Compliance APIs — aligned with UAE Open Finance and ISO 20022 standards.

---

## ⚙️ Features

- **API Catalog:** Discover mock KYC, KYB, Risk, and Lending APIs with live-style status, pricing, and ratings.  
- **API Documentation:** Swagger-style endpoint listings for developers.  
- **API Playground:** Simulated request/response interface using local JSON mock APIs.  
- **Trust Center (OpenID):** FAPI-compliant OAuth 2.0 / OpenID Connect endpoint simulation.  
- **AI Agents (In Development):** Preview of upcoming AI-powered assistants for fraud, AML, and KYC intelligence.  
- **Nth Exception / PaymentLabs branding:** Sapphire-teal-orange gradient, glass UI, and adaptive layout.
